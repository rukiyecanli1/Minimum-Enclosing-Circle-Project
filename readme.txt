Projeyi kendi noktalarınızla çalıştırmak için projenin bulunduğu klasördeki "noktalarr.txt" dosyasına kendi noktalarınızı yazabilirsiniz.
Noktalar arasına virgül koymadan yazmanız gerekmektedir.

Örnek:
5 -6  
4 2 
3 0

Proje C diliyle kodlanmış ancak grafik kütüphanesini sorunsuz çalıştırabilmek için .cpp dosya uzantısıyla açılmıştır.
